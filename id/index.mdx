WABeres adalah unofficial WhatsApp API gateway yang berjalan di atas protokol WhatsApp Web. 
Hubungkan nomor WhatsApp kamu dengan scan QR code — persis seperti WhatsApp Web — 
dan langsung dapatkan akses ke REST API, Webhook, dan Auto Reply.

<Note>
  Hanya butuh sedikit? WABeres menyediakan paket beli per kuota berdasarkan jumlah kuota pesan 
  mulai dari **Rp 10.000** tanpa masa kadaluwarsa.
</Note>

## Yang kamu dapatkan

**Tanpa registrasi.** Tidak ada akun, tidak ada form signup. Sistem otomatis membuat 
dashboard untukmu sesaat setelah pembayaran. Privasi pengguna dijaga penuh — tidak ada 
data identitas yang disimpan. [Pelajari cara kerjanya →](https://waberes.fredoronan.web.id#how)

**REST API + Webhook native.** Tersedia di semua paket termasuk beli per kuota. Cocok untuk 
membangun chatbot, integrasi AI/LLM, sistem notifikasi, atau otomasi apapun yang butuh 
WhatsApp sebagai channel-nya.

**Auto Reply.** Atur pesan balasan otomatis langsung dari dashboard — untuk toko tutup atau skenario apapun yang kamu butuhkan.

**Disposable tapi renewable.** Selesai pakai? Tinggalkan saja. Akun dihapus otomatis 
setelah kuota habis atau masa aktif berakhir. Untuk paket durasi, kamu bisa perpanjang 
masa aktif via API selama dalam rentang waktu 7 hari sebelum expired.

<Info>
  WABeres tidak menyimpan data identitas pengguna, namun log aktivitas sistem tetap 
  disimpan untuk keperluan debugging jika ada masalah selama kamu menggunakan platform ini.
</Info>

## 3 hal yang perlu kamu simpan

Setelah order, kamu akan menerima **invoice PDF** berisi tiga hal ini:

**Dashboard URL** — URL unik yang menjadi pusat kontrol akunmu. Pantau log API call, 
riwayat transaksi, dan aktivitas akun dari sini.

**API Key** — Sertakan di setiap request ke WABeres API.

**Secret Key** — Digunakan untuk verifikasi signature di setiap request dan validasi 
webhook yang dikirim server ke sistemmu.

<Warning>
  Simpan ketiganya di tempat yang aman. Dashboard URL berfungsi seperti password — 
  siapapun yang memilikinya bisa mengakses akunmu.
</Warning>
