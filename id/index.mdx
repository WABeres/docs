WABeres berjalan di atas protokol WhatsApp Web. Anda bisa langsung mengintegrasikan whatsapp ke sistem anda hanya dengan scan qr code persis seperti ketika anda 
menautkan akun whatsapp anda ke whatsapp web. Setelah berhasil terhubung anda akan mendapatkan fitur seperti auto reply, REST API dan Webhook.

<Note>
Hanya ingin mencoba saja? Atau sebenarnya hanya butuh sedikit saja?
WABeres menyediakan pembelian secara eceran (custom) berdasarkan jumlah pesan dengan harga mulai dari Rp. 10.000 saja.
</Note>

## Apa saja yang anda dapatkan?
1. **Tanpa akun, Tanpa registrasi, Tanpa login**. WABeres adalah platform integrasi whatsapp yang sama sekali tidak mengharuskan pengguna untuk melakukan registrasi
    akun terlebih dahulu pada platform. Hal ini bertujuan untuk mempermudah dan mempercepat proses pemenuhan kebutuhan integrasi whatsapp pada sistem anda serta
    mendukung penuh privasi pengguna dengan tidak menyimpan informasi terkait pengguna sama sekali. 
    Pelajari bagaimana cara kerjanya [disini](https://waberes.fredoronan.web.id#how)
2. **Bot/Chatbot AI/LLM**. Pada semua plan (termasuk versi eceran/custom), WABeres menyediakan fitur REST API + Webhook natively atau secara default. Anda bisa memanfaatkan fitur ini
    untuk membuat smart system anda sendiri dengan sangat mudah dan cepat dengan mengintegrasikan akun whatsapp anda seperti anda login ke whatsapp web. Semudah itu!
3. **Pemberitahuan singkat**. WABeres juga menyediakan pengaturan auto reply pada semua plan (termasuk versi eceran/custom). Fitur ini dapat anda memanfaatkan
    terutama bagi anda yang membutuhkan pesan balasan otomatis ketika toko libur atau tutup dan berbagai kebutuhan balasan otomatis lainnya.
4. **Disposables tapi renewable**. Sudah selesai? Tinggalkan saja. WABeres akan otomatis menghapus akun dashboard yang di berikan setelah masa aktif kadaluwarsa
    atau setelah kuota sudah habis. Hal ini dilakukan dengan tujuan untuk merawat server kami dan sekaligus mendukung privasi pengguna. **Namun**, jika anda memilih
    paket dengan tipe masa aktif (yang memiliki kadaluwarsa dan bukan berupa kuota) anda dapat **memperpanjang masa aktif** akun anda melalui API selama belum expired.

> _Walaupun WABeres adalah platform tanpa akun, tanpa registrasi dan tanpa login, log aktivitas sistem akan tetap di simpan untuk keperluan identifikasi
jika terdapat masalah pada pengalaman anda selama menggunakan platform ini. Hal ini dilakukan untuk mempertahankan kenyamanan anda dalam menggunakan WABeres
untuk sistem yang sedang anda bangun._


## Hanya ada 3 hal penting yang perlu di perhatikan

**Unique URL Dashboard** anda adalah pusat kontrol akun anda. Akun akan di generate otomatis oleh sistem. Anda tinggal mengunjungi url tersebut
untuk melihat semua aktivitas akun, riwayat transaksi, hingga log API call anda bisa di monitor disini.

**API Key**, seperti layanan API pada umumnya, anda harus menyimpan dan menyertakan API key ini di setiap request yang anda lakukan.

**Secret Key** digunakan untuk memverifikasi setiap request yang dilakukan baik dari anda atau webhook yang dikirim dari server ke sistem anda.
