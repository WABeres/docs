Setelah anda menyelesaikan panduan ini anda akan langsung mendapatkan akses ke dashboard beserta API Key, Secret Key dan pengaturan Webhook.

## Sebelum anda memulai

WABeres adalah platform tanpa akun. Setiap pembelian yang dilakukan user, sistem akan otomatis generate akun anonim untuk anda 
dengan detail paket sesuai yang anda pilih.

<Warning>
Terdapat file PDF yang akan otomatis terdownload. File tersebut berisi kredensial penting
seperti link untuk mengakses dashboard anda, API Key dan Secret Key. Jadi mohon untuk menjaga
dengan baik dokumen PDF tersebut karena hal tersebut merupakan tanggung jawab anda.
</Warning>
