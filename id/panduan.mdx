Setelah anda menyelesaikan panduan ini anda akan langsung mendapatkan akses ke dashboard beserta API Key, Secret Key dan pengaturan Webhook.

## Sebelum anda memulai

WABeres adalah platform tanpa akun. Setiap pembelian yang dilakukan user, sistem akan otomatis generate akun anonim untuk anda 
dengan detail paket sesuai yang anda pilih.

<Warning>
Terdapat file PDF yang akan otomatis terdownload. File tersebut berisi kredensial penting
seperti link untuk mengakses dashboard anda, API Key dan Secret Key. Jadi mohon untuk menjaga
dengan baik dokumen PDF tersebut karena hal tersebut merupakan tanggung jawab anda.
</Warning>

## Caranya
Silahkan kunjungi website WABeres [disini](https://waberes.fredoronan.web.id). Kemudian ikuti langkah - langkah berikut ini.

<Steps>
    <Step title="Memilih paket">
        Terdapat 3 paket durasi yang bisa anda pilih. Mulai dari paket 7 hari, 30 hari sampai 90 hari. Selain itu terdapat paket eceran juga
        yang dapat anda pilih dengan cara berikan spesifikasi jumlah kuota pesan yang anda butuhkan. 
        Harga akan dihitung sesuai jumlah pesan yang anda berikan tersebut.
    </Step>
    <Step title="Pembayaran">
        Ketika sudah memilih paket durasi ataupun eceran (per kuota pesan), anda akan diberikan sebuah peringatan bahwa anda akan langsung diarahkan ke
        halaman pembayaran. Kemudian jika anda menyetujui, silahkan pilih metode pembayaran favorit anda. 
    </Step>
    <Step title="Simpan dokumen PDF">
        Ketika berhasil melakukan pembayaran, sebuah dokumen PDF akan otomatis terdownload. Bila anda memakai perangkat mobile, klik download jika tidak
        secara otomatis terdownload.
        <Warning>
            **Mohon simpan dokumen PDF tersebut dengan baik karena akun anda hanya dapat di akses melalui link yang di berikan di dalam dokumen PDF 
            tersebut dan juga hati2 terkait siapa yang boleh mengetahui link tersebut**
        </Warning>
    </Step>
    <Step title="Selesai">
        Selamat! anda telah mendapatkan akses fitur integrasi whatsapp anda. Semoga sukses untuk sistem yang sedang anda bangun.
    </Step>
</Steps>
